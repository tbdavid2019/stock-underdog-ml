# 玄鐵重劍策略輸出解讀指南

> 本文件提供給 LLM 使用，說明如何解讀雙軌策略系統的輸出結果

---

## 📊 輸出結構

系統會產生三種結果：

### 1. 🗡️ 波段操作（玄鐵重劍策略）
- **持有期間**：2-4 週
- **篩選依據**：技術面 MA 均線
- **通過率**：15-25%

### 2. 🤖 短線操作（LSTM 預測）
- **持有期間**：1-7 天
- **篩選依據**：AI 預測漲幅
- **完成率**：100%（所有股票都會預測）

### 3. ⭐ 優先推薦（雙重符合）
- **持有期間**：2-4 週為主，可短線加碼
- **篩選依據**：同時通過技術面 + AI 預測
- **通過率**：10-20%

## 🔍 關鍵欄位解讀

### 回調類型欄位

#### **格式：** `MA60回調 (x.x%)` 或 `MA120回調 (x.x%)`

#### **MA60 vs MA120：**

| 回調類型 | 支撐強度 | 風險等級 | 適合投資者 | 說明 |
|---------|---------|---------|-----------|------|
| **MA60回調** | ⭐⭐⭐ 中期 | 🔥 較激進 | 積極型 | 60日均線，反應較快 |
| **MA120回調** | ⭐⭐⭐⭐ 中長期 | 🛡️ 較穩健 | 穩健型 | 120日均線，支撐更強 |

#### **百分比含義：**

| 回調百分比 | 位置關係 | 買入時機 | 風險評估 |
|-----------|---------|---------|---------|
| **+4% ~ +5%** | 股價在均線上方 4-5% | ✅ 好時機 | 趨勢已確認，可入場 |
| **+2% ~ +4%** | 股價在均線上方 2-4% | ✅ 好時機 | 健康回調後反彈 |
| **+0% ~ +2%** | 股價在均線上方 0-2% | ⭐ 最佳時機 | 幾乎貼合均線，支撐明確 |
| **-1% ~ 0%** | 股價略低於均線 | 🟡 觀察 | 剛跌破，可能反彈 |
| **-2% ~ -1%** | 股價在均線下方 1-2% | 🟡 等待 | 需確認是否止跌 |
| **< -2%** | 股價明顯跌破均線 | ❌ 避開 | 趨勢可能轉弱 |

**範例解讀：**
```
WFC: MA60回調 (+0.2%)
→ 股價幾乎完美貼合 60 日均線
→ ⭐⭐⭐ 最佳買點（支撐明確，風險低）

AMZN: MA120回調 (+3.4%)
→ 股價在 120 日均線上方 3.4%
→ ✅ 好買點（趨勢確認，支撐更強）

BKNG: MA120回調 (-2.2%)
→ 股價跌破 120 日均線 2.2%
→ ❌ 避開（跌破支撐，趨勢不明）
```

---

### PE/PB 基本面欄位

#### **PE（本益比）判讀：**

| PE 範圍 | 估值狀態 | 投資建議 |
|---------|---------|---------|
| **< 15** | 低估 | ✅ 價值投資好標的 |
| **15-25** | 合理 | ✅ 可以考慮 |
| **25-40** | 偏高 | 🟡 注意風險 |
| **> 40** | 高估 | ⚠️ 小心泡沫 |

#### **PB（股價淨值比）判讀：**

| PB 範圍 | 估值狀態 | 投資建議 |
|---------|---------|---------|
| **< 1.5** | 低估 | ✅ 可能被低估 |
| **1.5-3** | 合理 | ✅ 正常範圍 |
| **3-10** | 偏高 | 🟡 成長股或高估 |
| **> 10** | 極高 | ⚠️ 科技股或泡沫 |
| **負數** | 異常 | ❌ 負淨資產，財務問題 |

**特殊情況：**
- **PB 負數**（如 MCD: -101.46）→ 負淨資產，通常是大量回購或財務重組
- **PE 極高**（如 GEV: 105.26）→ 新創或高成長股，獲利還很少
- **PB 極高**（如 APP: 141.88）→ 輕資產科技公司

## 🎯 決策矩陣

### 優先推薦列表的解讀邏輯

#### **第一層篩選：LSTM 預測漲幅**

| LSTM 漲幅 | 優先級 | 操作建議 |
|----------|--------|---------|
| **> +5%** | 🔥 最高 | 強烈買入信號 |
| **+2% ~ +5%** | ✅ 高 | 可以買入 |
| **0% ~ +2%** | 🟡 中 | 觀察或小量 |
| **< 0%** | ⚠️ 低 | 謹慎或避開 |

#### **第二層篩選：回調類型**

```
MA60回調 (+0~2%) → 最佳入場點
MA60回調 (+2~5%) → 好入場點
MA120回調 (+0~2%) → 穩健入場點
MA120回調 (+2~5%) → 中性入場點
負數回調 → 降低優先級
```

#### **第三層篩選：估值**

```
PE < 20 且 PB < 3 → 加分
PE > 40 或 PB > 10 → 減分
PB 負數 → 警告
```

### 範例 1：理想標的
```

綜合評分: ⭐⭐⭐⭐ (4/5)
建議操作: 短線可積極，波段需謹慎（估值高）
持有期間: 1-2 週為主
```

### 範例 2：穩健標的
```
WFC (富國銀行)
✅ LSTM 預測: +8.29%      → 第一層：高優先級

✅ PE: 14.26              → 第三層：估值合理

綜合評分: ⭐⭐⭐⭐⭐ (5/5)
建議操作: 強烈推薦，波段+短線都適合

### 範例 3：需避開標的
```

❌ LSTM 預測: -6.90%      → 第一層：看空

⚠️  PE: 33.78             → 第三層：估值偏高

## 🎲 風險等級分類

根據綜合評分，將股票分為 5 個等級：

### S級 (⭐⭐⭐⭐⭐)
- LSTM > +5%
- 回調 0~2%
- PE < 25, PB < 3
- **範例：WFC, C**

### A級 (⭐⭐⭐⭐)
- LSTM > +3%
- 回調 0~4%
- PE < 40, PB < 10
- **範例：COF, CB, GEV**

### B級 (⭐⭐⭐)
- LSTM > +1%
- 回調 0~5%
- 估值可接受
- **範例：VRTX, AMGN**

### C級 (⭐⭐)
- LSTM < +1% 或負數
- 技術面尚可
- **範例：HCA, MDT**

### D級 (⭐)
- LSTM 負數
- 回調負數或估值異常
- **範例：BKNG, GE, IBM**

## 💡 特殊情況處理

### 情況 1：技術面好但 AI 看空
```
範例：IBM (MA60回調 +2.1%, LSTM -4.09%)

判斷邏輯：
- 技術面顯示買點
- AI 預測短期下跌
- 可能原因：基本面轉差、短期調整

建議：等待 AI 轉正，或降低倉位
```

### 情況 2：AI 看好但技術面轉弱
```
範例：某股 (MA60回調 -3%, LSTM +5%)

判斷邏輯：
- AI 看好上漲
- 但已跌破技術支撐
- 可能原因：超跌反彈機會

建議：等待重回均線上方再入場
```

### 情況 3：雙重看好但估值極高
```
範例：COF (LSTM +8.77%, PE 80.01)

判斷邏輯：
- 短期機會明確
- 長期估值風險高

建議：短線操作為主，設好停利點
```

## 🔧 LLM 使用注意事項

1. **始終檢查三個維度**：
   - LSTM 預測（短期動能）
   - 回調類型（入場時機）
   - PE/PB（估值風險）

2. **優先級順序**：
   - 雙重符合 > 單一策略
   - 正回調 > 負回調
   - 低估值 > 高估值

3. **風險警告觸發條件**：
   - PB 負數 → 必須警告
   - PE > 50 → 提醒高估值風險
   - 回調 < -2% → 提醒跌破支撐
   - LSTM < -5% → 提醒 AI 強烈看空

4. **不要只看單一指標**：
   - LSTM 高但估值高 → 短線可、波段險
   - 技術好但 AI 空 → 等待時機
   - 全部指標都好 → 強烈推薦

## 📚 術語表

| 術語 | 全稱 | 含義 |
|-----|------|------|
| **MA5** | 5日移動平均線 | 短期趨勢 |
| **MA60** | 60日移動平均線 | 中期趨勢（玄鐵策略主力） |
| **MA120** | 120日移動平均線 | 中長期支撐 |
| **MA250** | 250日移動平均線 | 長期價值中樞 |
| **PE** | Price-to-Earnings | 本益比（股價/每股盈餘） |
| **PB** | Price-to-Book | 股價淨值比（股價/每股淨值） |
| **LSTM** | Long Short-Term Memory | 深度學習模型 |
| **回調** | Pullback | 股價回到均線附近 |
| **雙重符合** | Dual Match | 同時通過技術面+AI預測 |
